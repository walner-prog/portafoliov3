Aprende HTML Básico

1. Introducción

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar el contenido de una página web mediante el uso de etiquetas que definen la función y presentación de diversos elementos, como textos, imágenes, enlaces y más. HTML proporciona la base estructural de una página, mientras que otros lenguajes como CSS y JavaScript se utilizan para estilos y comportamientos interactivos. Aquí tienes una estructura básica de un documento HTML:

                    
                        <!DOCTYPE html>
                        <html lang="es">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Mi primera página</title>
                        </head>
                        <body>
                            <h1>Hola, mundo!</h1>
                            <p>Este es un párrafo de ejemplo.</p>
                        </body>
                        </html>

2. Etiquetas Básicas

HTML utiliza etiquetas para estructurar el contenido. Algunas etiquetas básicas incluyen:

  • <h1> a <h6>: Encabezados
  • <p>: Párrafos
  • <a>: Enlaces
  • <img>: Imágenes
  • <ul>, <ol>, <li>: Listas
  • <div>: Contenedor genérico

3. Atributos

Las etiquetas pueden tener atributos que proporcionan información adicional. Por ejemplo, el atributo src en la etiqueta <img>:

                    
                        <img src="imagen.jpg" alt="Descripción de la imagen">

4. Enlaces

Crear enlaces es fundamental en HTML. Aquí tienes un ejemplo:

                    
                        <a href="https://www.ejemplo.com">Visitar Ejemplo</a>

5. Listas y Tablas

HTML te permite crear listas ordenadas, listas no ordenadas y tablas para organizar datos:

    
        <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
        </ul>

        <ol>
            <li>Primer paso</li>
            <li>Segundo paso</li>
            <li>Tercer paso</li>
        </ol>

        <table border="1">
            <tr>
                <th>Encabezado 1</th>
                <th>Encabezado 2</th>
            </tr>
            <tr>
                <td>Dato 1</td>
                <td>Dato 2</td>
            </tr>
        </table>

6. Formularios

Los formularios permiten a los usuarios enviar datos al servidor. Aquí tienes un formulario simple:

    
        <form action="/procesar" method="post">
            <label for="nombre">Nombre:</label>
            <input type="text" id="nombre" name="nombre">
            <br>
            <label for="email">Correo Electrónico:</label>
            <input type="email" id="email" name="email">
            <br>
            <input type="submit" value="Enviar">
        </form>

7. Multimedia

HTML permite incrustar contenido multimedia como audio y video:

    
        <audio controls>
            <source src="audio.mp3" type="audio/mp3">
            Tu navegador no soporta el elemento de audio.
        </audio>

        <video width="320" height="240" controls>
            <source src="video.mp4" type="video/mp4">
            Tu navegador no soporta el elemento de video.
        </video>

8. Comentarios y Entidades HTML

Puedes agregar comentarios en tu código HTML y utilizar entidades HTML para caracteres especiales:

    
        <!-- Esto es un comentario -->

        <p>Este es un ejemplo de <strong>texto fuerte</strong></p>

9. Etiquetas Semánticas

Las etiquetas semánticas como <header>, <footer>, y <section> proporcionan significado al contenido:

    
        <header>
            <h1>Mi Sitio Web</h1>
        </header>

        <section>
            <h2>Sección Principal</h2>
            <p>Contenido de la sección principal.</p>
        </section>

        <footer>
            <p>© 2024 Mi Sitio Web</p>
        </footer>

10. Enlaces Locales

Crea enlaces a otras páginas de tu propio sitio web:

    
        <a href="/pagina.html">Ir a Otra Página</a>

11. Enlaces Internos

Crear enlaces internos para navegar dentro de la misma página:


    <a href="#seccion1">Ir a la Sección 1</a>
